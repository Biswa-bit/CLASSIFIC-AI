# Supported Dataset Formats

## Current Version

CLASSIFIC-AI Version : v1.0

---

## Currently Supported Formats

| Format | Extension | Pandas Function |
|----------|-----------|----------------|
| CSV | .csv | pd.read_csv() |
| Excel | .xlsx | pd.read_excel() |
| Excel | .xls | pd.read_excel() |

---

## Format Recommendation

### CSV

Recommended for

- Small datasets
- Flat files
- Fast editing

Advantages

- Lightweight
- Easy to read
- Portable

---

### Excel

Recommended for

- Business datasets
- Reporting
- Multiple worksheets

Advantages

- Easy to use
- Business friendly
