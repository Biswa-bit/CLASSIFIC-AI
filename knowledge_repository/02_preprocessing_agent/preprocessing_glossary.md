# Preprocessing Glossary

---

# Overview

The Preprocessing Glossary is the centralized dictionary of preprocessing terminology used throughout the CLASSIFIC-AI framework.

Its purpose is to provide clear, consistent, and standardized definitions for preprocessing concepts, statistical terminology, machine learning techniques, and CLASSIFIC-AI recommendations.

The glossary serves three primary purposes:

- Educational reference
- Knowledge Repository asset
- RAG Knowledge Source

Whenever the CLASSIFIC-AI Copilot receives a definition-based question, this glossary becomes one of the primary knowledge sources.

---

# Purpose

The objectives of the glossary are:

- Standardize terminology
- Improve documentation consistency
- Support RAG retrieval
- Help new users learn preprocessing
- Improve explainability
- Provide interview preparation material

# Terminology

---

## Cardinality

### Definition

The number of unique values present in a categorical feature.

### Formula

```
Cardinality = Number of Unique Values
```

### Example

```
Country

India

USA

UK

India

USA

Cardinality = 3
```

### Used By

- High Cardinality Detection
- Encoding Recommendation

---

## Duplicate Row

### Definition

A row that contains exactly the same values as another row in the dataset.

### Example

| Name | Age |
|------|-----|
| John | 25 |
| John | 25 |

### Used By

- Duplicate Detection

---

## Missing Value

### Definition

A value that is unavailable or not recorded.

### Examples

```
NaN

NULL

None

Blank
```

### Used By

- Missing Value Detection

---

## Outlier

### Definition

An observation that differs significantly from most other observations.

### Common Detection Methods

- IQR
- Z-Score
- Isolation Forest

### Used By

- Outlier Detection
- Scaling Recommendation

---

## Encoding

### Definition

The process of converting categorical variables into numerical values.

### Common Methods

- Label Encoding
- One-Hot Encoding
- Frequency Encoding
- Target Encoding
- Embeddings

### Used By

- Encoding Recommendation

---

## Scaling

### Definition

The process of transforming numerical variables to a common scale.

### Common Methods

- StandardScaler
- MinMaxScaler
- RobustScaler
- MaxAbsScaler

### Used By

- Scaling Recommendation

---

## Text Feature

### Definition

A column containing free-form language rather than structured values.

### Examples

- Job Description
- Customer Review
- Product Description

### Used By

- Text Detection

---

## Boolean Feature

### Definition

A feature containing only two logical states.

### Examples

```
True / False

Yes / No

1 / 0
```

### Used By

- Boolean Detection

---

## Constant Feature

### Definition

A feature with only one unique value across the dataset.

### Formula

```
Unique Values = 1
```

### Used By

- Constant Feature Detection

---

## Identifier (ID)

### Definition

A feature used to uniquely identify records.

### Examples

- Customer_ID
- Employee_ID
- Order_ID
- Invoice_Number

### Used By

- ID Detection

---

## Recommendation Engine

### Definition

The intelligent decision-making component that combines all preprocessing results and generates preprocessing recommendations.

### Used By

Entire Preprocessing Agent

---

## Human-in-the-Loop (HITL)

### Definition

A governance approach where preprocessing recommendations are reviewed and approved by the user before execution.

### Benefits

- Transparency
- Explainability
- Business validation
- Reduced risk

---

## Dataset Health Score

### Definition

An overall quality score representing dataset readiness for machine learning.

### Generated By

Recommendation Engine

---

## Feature Engineering

### Definition

The process of creating new predictive variables from existing data.

### Examples

- Date decomposition
- Interaction features
- Polynomial features

---

## Feature Selection

### Definition

The process of selecting the most informative variables for model training.

### Benefits

- Faster training
- Reduced overfitting
- Better interpretability

---

## Embeddings

### Definition

Dense numerical vector representations used for text or high-cardinality categorical data.

### Examples

- Word2Vec
- GloVe
- Sentence Transformers
- BERT Embeddings

---

## TF-IDF

### Full Form

Term Frequency – Inverse Document Frequency

### Definition

A statistical method used to convert text into numerical features.

### Used By

Text Detection

---

## RobustScaler

### Definition

A scaling technique that uses the median and Interquartile Range (IQR), making it resistant to outliers.

### Recommended When

Dataset contains significant outliers.

---

## One-Hot Encoding

### Definition

An encoding technique that creates one binary feature for every category.

### Recommended When

Low-cardinality categorical variables.

---

## Frequency Encoding

### Definition

Each category is replaced by its frequency within the dataset.

### Recommended When

Medium-cardinality variables.

---

## Dataset Profiling

### Definition

The statistical analysis of a dataset after preprocessing.

### Next Agent

Data Profiling Agent
