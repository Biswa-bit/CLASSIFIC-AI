# Recommendation Engine

---

# Overview

The Recommendation Engine is the decision-making component of the CLASSIFIC-AI Preprocessing Agent.

Rather than automatically modifying datasets, the Recommendation Engine analyzes the outputs generated by every preprocessing module, evaluates dataset characteristics, and produces intelligent preprocessing recommendations for the user.

The Recommendation Engine follows the Human-in-the-Loop (HITL) philosophy of CLASSIFIC-AI. Every recommendation is presented to the user for review before any preprocessing action is executed.

---

# Purpose

The objectives of the Recommendation Engine are:

- Consolidate preprocessing results
- Generate intelligent preprocessing recommendations
- Explain every recommendation
- Improve dataset quality
- Support human decision making
- Prepare the dataset for downstream AI agents

The Recommendation Engine evaluates:

- Missing value percentage
- Feature data types

# Dataset Health Score

The Recommendation Engine calculates an overall Dataset Health Score.

Example

| Score | Dataset Status |
|--------|----------------|
| 90–100 | Excellent |
| 75–89 | Good |
| 60–74 | Fair |
| Below 60 | Needs Attention |

The score provides a high-level assessment of dataset readiness.

---

# Human-in-the-Loop

CLASSIFIC-AI does not automatically apply recommendations.

The user decides whether to:

- Accept all recommendations
- Accept selected recommendations
- Modify recommendations
- Reject recommendations
- Apply business-specific preprocessing

This ensures that preprocessing decisions remain aligned with business objectives.
