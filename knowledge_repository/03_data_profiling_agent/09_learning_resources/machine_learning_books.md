# Machine Learning Books

---

# Overview

This document provides a curated collection of machine learning books recommended for users, contributors, and developers of the CLASSIFIC-AI framework.

The selected books cover the complete machine learning lifecycle, including:

- Statistics
- Data Preprocessing
- Exploratory Data Analysis
- Machine Learning Algorithms
- Feature Engineering
- Explainable AI
- MLOps
- Deep Learning
- Enterprise AI

These resources support both theoretical understanding and practical implementation throughout the CLASSIFIC-AI architecture.

---

# Purpose

The objectives are to:

- Recommend high-quality learning resources
- Support continuous learning
- Improve machine learning expertise
- Provide references for enterprise AI
- Assist contributors to the CLASSIFIC-AI project
