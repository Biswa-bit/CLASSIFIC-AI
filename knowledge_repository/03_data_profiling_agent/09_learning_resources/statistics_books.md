# Statistics Books

---

# Overview

Statistics is the foundation of data science, machine learning, artificial intelligence, and business analytics.

The CLASSIFIC-AI framework relies extensively on statistical concepts throughout the machine learning lifecycle, including data profiling, exploratory data analysis, feature engineering, model evaluation, explainability, and simulation.

This document provides a curated collection of statistics books recommended for users, contributors, and developers working with CLASSIFIC-AI.

---

# Purpose

The objectives are to:

- Build a strong statistical foundation
- Improve analytical thinking
- Support explainable machine learning
- Enhance business decision-making
- Provide learning resources for contributors
